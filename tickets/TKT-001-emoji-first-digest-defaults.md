# TKT-001: Emoji-first digest defaults

- Status: todo
- Priority: P0

## Goal

Digest 기본 매칭 기준을 direct mention 중심이 아니라 watched emoji 중심으로 재정리한다.

## Why

- direct mention은 noise가 많음
- 사용자가 직접 남긴 emoji signal이 backlog 용도에 더 적합함

## Scope

- direct mention 포함 여부를 설정값으로 재검토
- digest 기본값을 emoji-first로 설계
- 관련 UX / help copy 수정

## Done when

- 기본 digest matching 전략이 문서화됨
- 설정값/기본값이 정리됨
- 관련 테스트 추가됨
